### make

#### build   
`make`
#### remove
`make clean`

### useage

`cal` 
> print calendar of this month
`.\cal `

`cal [year]`
> print calendar of this year
`.\cal 2018`

`cal [year] [month]`
> print calendar of [month] in [year]
`.\cal 2018 11`

`cal -m [month]`
> print calendar of [month] in this year
`.\cal -m 12`
