### 技术
React-native技术栈
### 功能描述
主要基于游戏的思想实现了一个GTD工具，用户给自己增加任务，完成的话可以有一个虚拟的钻石奖励。用这些虚拟奖励可以为自己“购买”一些对自己“不利”的“商品”，帮助用户实现自我管理。
